PM Pal - Documentation

Overview

PM Pal is a comprehensive project management assistant tool designed to help streamline communication and project tracking. The application consists of several key modules, each serving specific project management needs.


Figure 1: Main dashboard with the key features

Mail Mind Updated in ver. 1.2.4

The Mail Mind is an invaluable resource for anyone looking to improve their email communication. By providing sentiment analysis, key point extraction, action item identification, and calendar event creation, the Email Decoder helps users understand and manage their emails more effectively. Meanwhile, the Email Drafter ensures that responses are professional, clear, and tailored to the needs of the situation.

Together, these features empower users to navigate their email correspondence with confidence and efficiency.

Email Decoder - Key Features


Figure 2: The email Decoder interface
  1. Sentiment Analysis

    The Sentiment Analysis feature evaluates the emotional tone of the email. It categorizes the sentiment as positive, negative, or neutral, providing insights into the sender's feelings and intentions. This can help you gauge the urgency or importance of the email and tailor your response accordingly.

  2. Severity Flagging

    Automatically flags emails based on impact and urgency (Coming Soon)

  3. Key Points

    This feature extracts the main ideas or concerns from the email, summarizing the essential information for quick reference. By highlighting key points, users can focus on the most relevant aspects of the communication without having to sift through the entire message.

  4. Action Items

    The Action Items feature identifies specific tasks or requests mentioned in the email. It helps users to quickly recognize what needs to be done in response to the email, ensuring that important tasks are not overlooked.

  5. Calendar Events

    This feature scans the email for any mentions of dates, times, or events that may require scheduling. It allows users to easily identify and create calendar events based on the information provided in the email, streamlining the process of managing appointments and deadlines.

Email Drafter - Key Features


Figure 3: The email Drafter interface
  1. Professional Positive Response

    The Email Drafter can generate a response that conveys a positive and professional tone. This feature is ideal for situations where you want to maintain a cordial relationship with the sender while addressing their concerns or inquiries.

  2. Enhanced Drafting

    If you have a draft email that requires refinement, this feature enhances the existing content while preserving the core message. It ensures that the response maintains a professional tone and clarity, making it suitable for formal communication.

  3. Hint-Based Response

    Using hints provided by the user, this feature crafts a response that is professional, clear, and addresses all specified points. It ensures that the reply is comprehensive and meets the expectations of the recipient, facilitating effective communication.

    How does this work?

    A professional positive response is crafted if both draft and hints are not provided. For the remaining two, choose either of them or both to generate a custom draft.

The Email Alchemist: Intelligent Email Analysis Tool Updated in v1.2.4

Product Documentation

The e-Alchemist is a sophisticated email analysis tool that revolutionizes project management communication. By leveraging advanced AI technology, it transforms complex email threads into clear, actionable insights, saving valuable time for project managers and team members.

It is designed to empower users with advanced tools for email management and analysis, ensuring secure integration, intuitive search capabilities, and comprehensive insights that drive productivity and informed decision-making.

    Secure Gmail Integration
    1. One-click Gmail Authentication: Users can easily authenticate their Gmail accounts using the OAuth2 protocol, ensuring a secure connection.
    2. Real-time Mailbox Status Indicator: A navigation bar displays the current status of the mailbox, providing instant feedback on connectivity.
    3. Secure Token Management: The system manages authentication tokens securely, with automatic refresh capabilities to maintain session integrity.
    4. Persistent Authentication: Users remain logged in across sessions, eliminating the need for repeated logins.
    Intuitive Search
    1. Standard Gmail Search Support: The feature supports standard Gmail search formats, enhancing user familiarity.
    2. Automatic Search Identifiers: Automatically adds search identifiers such as "subject:", "from:", and "label:" to streamline the search process.
    Advanced Email Retrieval
    1. Customizable Date Range Selection: Users can specify date ranges for fetching emails, allowing for targeted retrieval.
    2. Keyword-based Search Functionality: Facilitates searching for specific keywords within emails to quickly locate relevant information.
    3. Support for Multiple Email Formats: Capable of handling various email formats, including plain text and HTML.
    4. Intelligent Content Extraction: Extracts and cleans email content intelligently, ensuring clarity and relevance.
    Comprehensive AI Analysis
    1. Context-aware Interpretation: The AI analyzes email content with context in mind, providing deeper insights.
    2. Multi-email Thread Analysis: Capable of analyzing entire email threads to capture the full conversation context.
    3. Identification of Project-critical Information: Highlights key information that is critical to ongoing projects.
    4. Priority and Urgency Assessment: Evaluates the urgency of emails to help prioritize responses.
    5. Relationship Mapping: Maps relationships between communications to identify key stakeholders and their interactions.
    Structured Output Format
    1. Executive Summary: Provides a concise summary of key points from the analyzed emails.
    2. Time-sensitive Action Items: Lists actionable items with deadlines to ensure timely follow-up.
    3. Critical Decisions and Implications: Highlights important decisions made within the email threads and their potential impacts.
    4. Project Impact Assessment: Evaluates how communications affect ongoing projects.
    Interactive User Interface
    1. Dual-view Toggle: Users can switch between raw email views and insights for a comprehensive understanding.
    2. Progress Indicators: Displays progress during analysis to keep users informed.
    3. User-friendly Error Handling: Provides clear and helpful error messages to guide users in troubleshooting.
    Enhanced Productivity Features
    1. One-click Copying: Easily copy analysis results for quick sharing or documentation.
    2. Original Email Content Preservation: Maintains the original email content for reference alongside analysis.
    3. Export Functionality: Allows users to export analysis results for documentation purposes.
    4. Batch Processing: Supports the analysis of multiple emails simultaneously, enhancing efficiency.

User Experience

The intuitive workflow guides users through three simple steps:

  1. Authentication: Secure one-click Gmail connection with visual status confirmation
  2. Email Selection: Flexible date range and keyword filtering for relevant emails
  3. Analysis: Automated processing with real-time progress updates and clear results presentation

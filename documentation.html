PM Pal - Documentation

Overview

PM Pal is a comprehensive project management assistant tool designed to help streamline communication and project tracking. The application consists of several key modules, each serving specific project management needs.


Figure 1: Main dashboard showing all available features

Email Decoder Updated in v1.1

The Email Decoder module helps analyze and interpret email communications, providing insights and suggested responses.

Key Features

  1. Email Analysis Options
    • Draft a positive response to the email
    • Draft an email using provided hints
    • Improvise existing drafts

    Figure 2: Email analysis options and interface
  2. Severity Flagging

    Automatically flags emails based on impact and urgency (Coming Soon)

The Email Alchemist Updated in v1.2

A powerful tool for transforming and analyzing email communications.

Authentication

  1. Email verification through OTP
  2. Mailbox authentication using OAuth
  3. Secure token management

Email Analysis Features

  • Advanced search capabilities
  • Customizable insights
  • Multiple output formats:
    • Key Points
    • Tabular Format
    • Executive Summary
